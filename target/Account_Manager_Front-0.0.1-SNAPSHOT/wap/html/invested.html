对投资人的忠告和提醒。

我们提醒每一位投资者：市场有风险，投资需谨慎。在投资前需要对项目的基本信息进行了解后再进行理性和谨慎的投资，建议在不影响生活的情况下合理配置投资资金。

左右逢园平台作为网络借贷信息中介服务平台，为用户提供信息发布、资料审核、借贷撮合、风控管理等法律法律法规所允许的业务，并确保在平台上所产生的借贷关系具有法律效力。平台将努力降低投资者的投资风险，设置投资保障措施，但不对融资项目进行担保和代偿。

投资人需要具备什么样的条件？

投资者需要年满18岁，有通过合法途径获得的且可自由支配的资金，并且对网络借贷的知识和风险有一定了解。

进行投资的前提条件。

用户在左右逢园投资需要先完成注册操作，并绑定银行卡后才能进行投资。

投资项目的收益及期限范围。

因项目不同，年化收益及投资期限也会有所不同，每个项目在展示过程中均会对投资预期收益及投资期限进行展示告知。

投资后如何计息还款？

用户投资成功后，从项目结标日（即借款资金划转到融资方账户之日）起开始计算收益，遇到周末和法定节假日，则顺延至节后第一个工作日。项目采用按月付息到期还本的方式进行还款。

导致投标失败的原因是什么？

投标失败可能源于以下几种原因：1.投资人账户内资金不足，需要充值；2.在完成投资操作前此项目已经满标；3.投标时间已截止；4.流标。

借款人逾期了怎么办？

如果借款人发生逾期，为借款项目提供担保的担保公司在24小时内进行本息垫付，在担保公司无力偿还的情况下，启用合作担保公司保证金进行垫付，保障投资人的资金安全。

投标成功后能否撤回投资？

投标成功后，投资资金将被系统冻结，待满标后直接转入借款人账户，此时进入还款期。在这一过程中用户所投资金将无法撤回。因此，请您在投资前确认好所投资项目的各项信息及投资金额。

投资期满后，投资本金将回到您的资金账户。

投资者的收益如何保障？

作为信息中介平台，主要职能是交易撮合，同时设置多项风险防控措施，对投资者的收益进行保障。

（1）电子合同，项目结标后平台会产生具有法律效力的《借款协议》，明确投融资双方的债权债务关系；

（2）第三方担保，实力担保机构提供连带责任担保，逾期由担保公司提供全额本息垫付；

（3）其它担保措施，为保证资金安全，每个项目都会设置多项风险防控措施，具体参见“项目信息”中的“风险措施”项。

回款日资金何时到账？

投资人的利息或本金在回款日当天24点前到账均为正常还款。受借款人还款时间、资金存管银行业务办理时间等因素的影响，具体还款时间会有所不同。
